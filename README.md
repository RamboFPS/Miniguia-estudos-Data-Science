## 📖 Miniguia de Estudo 

### Resumos Estruturados
- Principais conceitos de estatística inferencial.  
- Diferença entre regressão e classificação.  
- Ciclo de vida de um projeto de dados.  

### Glossário
- **EDA (Exploratory Data Analysis):** análise exploratória dos dados.  
- **Overfitting:** quando o modelo se ajusta demais ao conjunto de treino e perde generalização.  
- **MLOps:** práticas para colocar modelos em produção de forma escalável.  

### Prompts Reutilizáveis
- *"Explique [conceito] em linguagem simples e com exemplos."*  
- *"Mostre um caso prático de [técnica] aplicada em dados reais."*  
- *"Crie um resumo em tópicos sobre [tema]."*
